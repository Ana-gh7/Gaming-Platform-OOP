# Gaming Platform - OOP Project

A Java Object-Oriented Programming project that simulates an online gaming platform.

## Features

- User management (Player, Administrator)
- Game management (Singleplayer and Multiplayer)
- Game reviews and ratings
- Personal game library
- Shopping cart and orders
- Payment processing (Card and PayPal)
- File operations for saving and loading games

## Design Patterns Used

- Factory Pattern
- Strategy Pattern
- Iterator Pattern

## Main Concepts

- Inheritance
- Abstraction
- Encapsulation
- Polymorphism
- Interfaces
- Custom Exceptions
- Collections

## Class Structure

### Users
- User (abstract)
- Player
- Administrator

### Games
- Game (abstract)
- SingleplayerGame
- MultiplayerGame

### Payments
- PaymentMethod (interface)
- CardPayment
- PayPalPayment
- PaymentProcessor

### Reviews
- Review
- Reviewable (interface)

### Collections
- GameLibrary
- ShoppingCart
- Order
